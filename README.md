# 🍺 Таверна Легенд (Tavern Legends)

Браузерная коллекционная карточная игра, вдохновлённая Hearthstone. Чистый
HTML/CSS/JS без зависимостей и сборки — открой `index.html` и играй.

**▶️ Играть: https://vraiuk.github.io/hearthstone-clone/**

![genre](https://img.shields.io/badge/genre-CCG-orange) ![deps](https://img.shields.io/badge/dependencies-0-brightgreen) ![lang](https://img.shields.io/badge/lang-RU-blue)

## Возможности

- **Полноценный движок боя**: мана-кристаллы 1→10, существа, заклинания, оружие,
  броня, усталость (fatigue), монетка второму игроку.
- **Ключевые слова**: Провокация, Рывок, Натиск, Божественный щит, Похищение
  жизни, Неистовство ветра, Яд, Боевой клич, Предсмертный хрип, Сила заклинаний,
  Заморозка, Немота.
- **4 класса** со своими силами героя и колодами: Маг 🧙, Воин ⚔️, Жрец ✨, Охотник 🏹.
- **75+ карт** — свой арт-стиль (CSS + эмодзи), никаких внешних ассетов.
- **Кампания на 10 боёв** с боссами-модификаторами (стартовая броня, +10 здоровья,
  существа на столе с 1-го хода, двойная сила героя у финального босса).
- **Прогрессия**: золото за победы, разблокировка карт, крафт за золото,
  редактор колод, сохранение в localStorage.
- **ИИ-противник** с тремя уровнями: играет по кривой маны, торгует существами,
  добивает при летале.

## Запуск локально

```bash
# любой статический сервер, например:
python3 -m http.server 8000
# → http://localhost:8000
```

Подойдёт любой статический хостинг — сборка не нужна.

## Тесты

```bash
node tests/smoke.mjs
```

Прогоняет 40 полных игр ИИ против ИИ с проверкой инвариантов + юнит-проверки
механик (щит, хрипы, броня, провокация, яд, неистовство ветра и др.).

## Архитектура

```
js/
  engine/   — правила игры: game.js (ходы, бой, смерти), effects.js (декларативные эффекты)
  data/     — cards.js (база карт), decks.js (стартовые колоды), campaign.js (кампания)
  ai/       — эвристический ИИ-противник
  ui/       — battle.js (экран боя), menus.js (меню/кампания/коллекция), cardRender.js
  state/    — save.js (прогрессия в localStorage)
```

Карты — чистые данные с декларативными «оп-кодами» эффектов; движок резолвит их
в одном месте. Это делает контент легко расширяемым и тестируемым.
